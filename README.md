# 🏥 Hospital Appointment System

A Java-based desktop application designed to simplify appointment management between patients and healthcare providers.

## Overview

The Hospital Appointment System allows patients to create accounts, log in securely, browse doctors by specialty, schedule appointments, view appointment details, track appointment status, and cancel appointments through an easy-to-use interface.

## Features

- User Registration & Authentication
- Patient Dashboard
- Doctor Directory
- Specialty-Based Doctor Filtering
- Appointment Booking
- Appointment Tracking
- Appointment Cancellation
- Oracle Database Integration

## Technologies Used

- Java
- Java Swing
- JDBC
- Oracle Database
- Object-Oriented Programming (OOP)

## System Modules

### Patient Module
- Register Account
- Login
- Browse Doctors
- Book Appointments
- View Appointments
- Cancel Appointments

### Appointment Management
- Schedule Appointments
- View Appointment Details
- Track Appointment Status
- Manage Bookings

### Doctor Directory
- View Available Doctors
- Filter by Specialty
- Display Doctor Information

## Database Design

The system is built around four main entities:

- Users
- Patients
- Doctors
- Appointments
